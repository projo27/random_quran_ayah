This is a next.js project to get random daily quran ayah
the goals of this apps was, to helps you recite quran
and make it easy for you to share the quran, may you have
reward from ALLAH


credits : https://api.alquran.cloud/v1  



## Getting Started

``` bash
npm run dev
```
